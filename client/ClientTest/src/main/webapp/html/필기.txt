HTML, Hyper Text Markup Language
- 하이퍼 텍스트 마크업 언어
- Hyper Text: 링크(Link)
- Markup Language: 태그(Tag)를 사용해서 문법을 표기하는 언어
    - SGML, XML, HTML, XHTML 등..
- 프로그래밍 언어(X) -> 표현 언어(O): 브라우저 화면에 무언가를 출력하는 기능만 있는 언어
- 난이도 하 -> 개발자 입장 -> 중하
- http://w3.org, http://w3c.org > The World Wide Web Consortium (W3C) > 웹 관련 기술 표준화 작업
- HTML 1.0 ~ HTML5
- 수업: HTML 4.01 + XHTML 1.0 + HTML5

브라우저, Web Browser
- 웹 서버에 접속해서, 필요한 자원(HTML페이지, 이미지, 동영상 등)을 다운로드 받아서 처리 후에 화면에 결과를 출력하는 프로그램
- HTML 언어를 실행하는 환경(HTML 언어 분석 + 파싱 + 실행 + 결과)
- 자바에 비교하면 JDK + JRE와 유사
